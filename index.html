About me

Hello, I am Arjan den Os.

After a life-changing experience I have decided to go do something that seems interesting Ã¡nd fun to me: making websites!
First I've learned the basics of correct HTML and had an introduction to CSS via FreeCodeCamp and The Odin Project. Now I was able to make a (static) webpage, the most logical step seemed to be to learn how to design a webpage before producing it. I then watched a plethora of tutorials about design and how to work with Figma. Color-theory, contrast-ratios, alignment, visual hiarchy, typography... But I realised that wasn't where my major interest is.
After finishing a Computer Science course from Harvard professor David J. Malan a couple puzzlepieces fell into place and a whole new world opened for me. After that I've decided to fully focus myself on (front-end) webdevelopment. At Udemy I first started with a 2-part course DOM-manipulation. Next was an advanced CSS/SASS course. And after that I've finished a most complete JavaScript course
Everyday I'm getting more comfortable in working with the web trio HTML/CSS/JS. Evaluating the start of my developer journey, this is the result after approximately 6 months: 3 projects and this portfolio as demonstration of my current progress.
My current tools:

In January 2022 I've started a most complete JavaScript course. From the basics of JS and the DOM to working with classes, API's and async behaviour. Developer tools, paradigms, modular building, how the single-threaded JS works "under the hood"; it all has been explained. Too much to list honestly. And everything in the modern ES6 ofcourse!

Via Udemy I have completed an advanced CSS/SASS course with flexbox and grid as basis, but also 'older' techniques have been explained. Trying to create fully responsive layout with as little media-query's as possible was an eye-opener and sometimes a challenging puzzle. Working with SASS and compiling with NPM was a good introduction to professional developing.

I started with FreeCodeCamp and The Odin Project to learn the basics of HTML. Forms, images, links etcetera. Via a11yproject I eventually learned the importance of semantically correct HTML and how the accessibility-tree node works.
