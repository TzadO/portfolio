About me

Hello, I am Arjan den Os.

After a life-changing experience, I've decided to go do something that seems exciting to me: making websites!
First, I've learned the basics of HTML and CSS via FreeCodeCamp and The Odin Project. Now I was able to make a web page, the most logical step seemed to learn how to design a webpage before producing it. I then focused for a while on studying color theory, contrast-ratios, alignment, visual hierarchy, typography... etcetera. But I realized that design is not where my major interest is.

After finishing a Computer Science course from Harvard professor David J. Malan, a whole new world opened up for me. I then decided to fully focus on (front-end) web development. Learning JavaScript before learning a framework seemed the most logical thing to do. At Udemy I first started with a 2-part course DOM-manipulation. Next I finished a most complete JavaScript course and an advanced CSS/SASS course. Every day I'm getting more comfortable in working with the web trio HTML/CSS/JS.

From the basics of JS and the DOM to working with classes, API's and async behaviour. Developer tools, paradigms, modular building and how the single-threaded JS works "under the hood"; it all has been explained. Too much to list, honestly. And everything in the modern ES6 synthax of course!

Via Udemy I have completed an advanced CSS/SASS course with flexbox and grid as basis, but also 'older' techniques have been explained. Trying to create a fully responsive layout with as little media-query's as possible was an eye-opener and sometimes a challenging puzzle. Working with SASS and compiling with NPM was a good introduction to professional developing.

I started to learn the basics of HTML via FreeCodeCamp and The Odin Project. Forms, images, links etcetera. Later I learned the importance of semantically correct HTML for accessibility and SEO.

Currently...

...I'm learning the framework React and I'm willing to learn other frameworks.
In general, I'm trying to get more comfortable in the use of NPM, Git and SASS.
Every day I continue to broaden my knowledge of CSS and JS.
I would like to start working in a professional environment as the next step, to be able to learn from experienced developers.
Eventually I would like to start a course on Node.JS, so can practice projects on my Raspberry Pi and learn more about the back-end.
For now, the focus is 100% on front-end.
